# 🎵 Перенос треков в Яндекс Музыку

Полувтоматический скрипт для удобного переноса вашей музыкальной библиотеки из ВКонтакте (или других сервисов) в Яндекс Музыку. 

## 📂 Структура

- `import_music.py` — основной скрипт;
- `music.txt` — текстовый файл с вашими треками для переноса (создайте его);
- `failed_tracks.txt` — файл с треками, которые скрипт не смог найти (появится после работы скрипта).

---

## 📝 Как получить список треков (music.txt)

### VK Музыка
1. Зайдите в раздел **«Моя музыка»** с компьютера.
2. Начните выделять треки мышком (сверху вниз).
3. Скопируйте выделенное сочетанием клавиш `Ctrl + C` (или `Cmd + C` на Mac).
4. Вставьте (`Ctrl + V`) всё в файл `music.txt` в этой папке.

> 💡 Скрипт автоматически обрабатывает треки **с конца списка**.

---

## 🚀 Запуск скрипта

1. **Получите свой токен Яндекс Музыки**  
   Скрипту нужен доступ к вашему аккаунту. Способы получения токена можно найти [здесь (на GitHub)](https://github.com/MarshalX/yandex-music-api/discussions/513).  
   *(На 17.04.2026 хорошо работает [способ с троттлингом](https://github.com/MarshalX/yandex-music-api/discussions/513#discussioncomment-2729781)).*

2. **Установите зависимости**  
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите скрипт**  
   Запустите скрипт следующей командой, подставив свой токен вместо `YOUR_TOKEN_HERE`:
   ```bash
   python import_music.py music.txt --token "YOUR_TOKEN_HERE"
   ```
   Найденные треки будут автоматически добавляться в плейлист **"Мне нравится"**.

4. **Ожидайте завершения работы скрипта**  
   Для каждого трека будет выведен статус. В случае, если песня не найдена, в конце будет создан файл `failed_tracks.txt` для ручного поиска.

---

## ⚙️ Формат music.txt и примечания

Скрипт читает файл построчно. Любые текстовые строки (исполнитель, название трека, прочая информация) объединяются в один запрос для поиска. 

Разделителем между разными треками служит **строка с продолжительностью песни** (например, `3:44` или `12:05`). Поэтому каждый блок песни **обязательно должен заканчиваться таймкодом!**

**Пример правильного файла:**
```text
look the eyes, kiramo
lost on you x hoodtrap
3:44
i hate x rinoplastika
itgmqprod, teiqol
i hate x rinoplastika mylancore
2:47
```

**Либо так:**
```text
look the eyes, kiramo lost on you x hoodtrap 
3:44
i hate x rinoplastika itgmqprod, teiqol i hate x rinoplastika mylancore 
2:47
```
